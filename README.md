# Dashboard de Análise de Veículos Usados

## Descrição do Projeto
Este projeto é um dashboard web interativo desenvolvido com Streamlit para análise de dados de anúncios de vendas de carros usados. O aplicativo permite visualizar dados através de histogramas e gráficos de dispersão com controles interativos.

## Funcionalidades
- Visualização de histogramas interativos
- Gráficos de dispersão para análise de correlações
- Botões e caixas de seleção para controle das visualizações
- Interface web responsiva

## Como Executar o Projeto

### Pré-requisitos
- Python 3.7+
- Bibliotecas listadas em `requirement.txt`

### Instalação
1. Clone este repositório:
git clone https://github.com/deyvlablemandrade/Projeto_Sprint_5.git

    Instale as dependências:
    pip install -r requirement.txt

    Execute o aplicativo:
    streamlit run app.py

## Tecnologias Utilizadas
- Streamlit: Framework para criação do dashboard web
- Pandas: Manipulação e análise de dados
- Plotly Express: Criação de gráficos interativos
- Render: Plataforma de deploy

## Dataset
O projeto utiliza dados de anúncios de vendas de carros usados, contendo informações sobre:
- Modelo e marca dos veículos
- Preço e quilometragem
- Ano de fabricação
- Condição do veículo
